MonoDevelop.Tizen

MonoDevelop add-in which adds support for developing and deploying LGPLv2-compliant applications using Mono for Tizen.
Download this project as a .zip file Download this project as a tar.gz file

Open the IDE

  • Open MonoDevelop or Xamarin Studio.

Install and enable the Tizen development add-in

  • In the Add-in Manager dialog, you will see a Mobile Development tree-view container. Click on it to expand it.
  • You will see the Tizen development plugin. Click on it, then click on Install... and then the Install button again on the modal dialog which appears.
  • You will see the installation proceeding. It only takes a few seconds.
  • In the right hand panel you will now see an Enable button. Click it and icon in the left hand panel will switch from Gray to Green.
  • Close the Add-in Manager dialog.

Getting future add-in updates?

  • Xamarin Studio | Check for Updates... will automatically pick up any future changes to the add-in.
